[CRM APP接口定义二阅读]

1:公司公告发布[只有管理员可以操作]

接口说明 :公司公告发布

请求地址 /api/v1/announcement/new 类型
请求参数 title 公告标题 string
请求参数 content 公告内容 string
返回值 result 1 操作成功 0 操作失败 int
data json
