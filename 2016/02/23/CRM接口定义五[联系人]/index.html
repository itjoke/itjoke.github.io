CRM APP接口定义五[圈子-平台联系人]

发表于 2016-02-23   |  
  • 联系人内分为3种数据，1是好友关系，2在销售连但不是好友关系，3仅仅只是录入的联系人，并未在销售连注册。

    联系人信息目前是在数据库中存一个大的字段，目前作为中间过渡。点赞信息列表目前也是一个大的字段中，目前也是作为中间过渡方案。

1:圈子信息列表

接口说明 :获取圈子信息列表-带5条评论

请求地址 /api/v1/circle/list 类型
请求参数 page 当前页 int
请求参数 pagenum 一页包含的数据条数 int
返回值 result 1 操作成功 0 操作失败 int
data json

内容暂定

2:圈子信息评论获取

接口说明 :圈子信息评论

请求地址 /api/v1/circle/comment 类型
请求参数 page 当前页 int
请求参数 pagenum 一页包含的数据条数 int
返回值 result 1 操作成功 0 操作失败 int
data json

内容暂定

3:圈子信息发表评论

接口说明 :圈子信息发表评论

请求地址 /api/v1/circle/comment/post 类型
请求参数 postid 主贴id int
请求参数 commentid 评论id int
请求参数 cotent 评论内容 string
返回值 result 1 操作成功 0 操作失败 int
data json

内容暂定

4:圈子信息点赞

接口说明 :圈子信息点赞

请求地址 /api/v1/circle/comment/favour 类型
请求参数 postid 主贴id int
请求参数 commentid 评论的id int
返回值 result 1 操作成功 0 操作失败 int
data json

内容暂定

5:圈子信息删除（只有发布者可以删除）

接口说明 :圈子信息删除

请求地址 /api/v1/circle/comment/del 类型
请求参数 postid 主贴id int
请求参数 commentid 评论id int
返回值 result 1 操作成功 0 操作失败 int
data json

内容暂定

6:获取平台我的个人信息

接口说明 :获取墙外我的个人信息(请求参数默认带uid)

请求地址 /api/v1/u/info 类型
返回值 result 1 操作成功 0 操作失败 int
data json

7:修改平台我的个人信息

接口说明 :修改墙外我的个人信息(请求参数默认带uid)

请求地址 /api/v1/u/edit 类型
请求参数 editval 参看请求参数 json
返回值 result 1 操作成功 0 操作失败 int
data json

请求参数为接口6返回的对象格式（剔除未修改的参数）

返回值

8:我加入的组织列表

结合userinfo表中的comidlist和organization表信息生成返回值

请求地址 /api/v1/organization/list 类型
返回值 result 1 操作成功 0 操作失败 int
data json

9:创建组织【重点】

请求地址 /api/v1/organization/new 类型
请求参数 name 组织（公司）名称 string
请求参数 desc 组织（公司）描述 string
返回值 result 1 操作成功 0 操作失败 int
data json

10:添加联系人

手机号和姓名是必填项

请求地址 /api/v1/contact/add 类型
请求参数 name 姓名 string
请求参数 organization 公司名 string
请求参数 title 职位 string
请求参数 dept 部门 string
请求参数 mobile 手机 string
请求参数 tel 电话 string
请求参数 mail 邮件 string
请求参数 address 地址 string
请求参数 website 网址 string
请求参数 remark 备注 string
返回值 result 1 操作成功 0 操作失败 int
data json

11:修改联系人

手机号和姓名是必填项

请求地址 /api/v1/contact/edit 类型
请求参数 jsonobject 被修改的字段名和值 string
返回值 result 1 操作成功 0 操作失败 int
data json

12:获取联系人列表

需要区分在本app注册过和未注册过的状态

请求地址 /api/v1/contact/list 类型
请求参数 updatetime 获取数据中的最新更新时间 int
返回值 result 1 操作成功 0 操作失败 int
data json

13:搜索用户

用户名搜索暂不实现

请求地址 /api/v1/contact/search 类型
请求参数 mobile 手机号 string
请求参数 name 用户名 string
返回值 result 1 操作成功 0 操作失败 int
data json

14:添加用户为好友

请求地址 /api/v1/contact/friend/add 类型
请求参数 fid 用户id int
返回值 result 1 操作成功 0 操作失败 int
data json

15:新用户添加请求处理

请求地址 /api/v1/contact/add/do 类型
请求参数 fid 用户id int
请求参数 action 是否接受 bool
返回值 result 1 操作成功 0 操作失败 int
data json

16:新朋友状态列表

包括添加其他好友和被其他用户添加为好友状态

请求地址 /api/v1/contact/new/list 类型
返回值 result 1 操作成功 0 操作失败 int
data json

17:圈子信息发布

接口说明 :圈子信息发布

请求地址 /api/v1/circle/new 类型
请求参数 content 公告内容 string
请求参数 picurl 图片url string
返回值 result 1 操作成功 0 操作失败 int
data json
