[CRM APP接口定义八工作]

1:获取工作列表

接口说明 :获取工作列表

返回工作列表需要带权限信息，地理位置信息，图片信息

请求地址 /api/v1/work/list 类型
请求参数 type 工作类型(我的工作，下属的工作，参与的工作，全部工作) string
请求参数 time 时间戳 string
返回值 result 1 操作成功 0 操作失败 int

2:工作搜索

接口说明 :按关键字搜索工作信息

返回工作列表需要带权限信息，地理位置信息，图片信息

请求地址 /api/v1/work/search 类型
请求参数 type 工作类型(我的工作，下属的工作，参与的工作，全部工作) string
请求参数 key 搜索关键字 string
返回值 result 1 操作成功 0 操作失败 int
data json

例子

3:删除某条工作信息

接口说明 :删除

请求地址 /api/v1/work/del 类型
请求参数 id 工作编号 string
返回值 result 1 操作成功 0 操作失败 int
data json

例子

4:发布工作

接口说明 :发布工作(工作内容必填)

请求地址 /api/v1/work/publish 类型
请求参数 type 工作类型 int
请求参数 worktime 工作时间 timestamp
请求参数 lal 签到地点（经纬度坐标） long
请求参数 address 地理位置信息 string
请求参数 content 工作内容 string
请求参数 picurl 图片（最多9张） string
请求参数 vediourl 视频 string
请求参数 radiourl 音频 string
请求参数 fileurl 文件 string
请求参数 customerlist 关联客户列表 string
请求参数 stafflist 关联同事列表 string
返回值 result 1 操作成功 0 操作失败 int
