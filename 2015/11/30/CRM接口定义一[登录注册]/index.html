[CRM APP接口定义一]

User Agent
需要将一下数据拼接成一个字符串填到UA中:
时间:YYMMdd
Encoding:
Locale:
Lang:
DeviceId:
UDID:
DeviceBrand:
DeviceModel:
DeviceVersion:
ClientType:

ScreenHeight:

token没有失效期，跟设备id绑定，除非更换设备，否则不需要重新登录获取token

1:登录接口

接口说明 :用户登录时调用

请求地址 /api/v1/u/login 类型
请求参数 mobile 手机号 String
pwd 密码 String
返回值 result 1 操作成功 0 操作失败 int
data json

2:注册接口

接口说明 :用户注册时调用

请求地址 /api/v1/u/reg 类型
请求参数 mobile 手机号 String
pwd 密码 String
返回值 result 1 操作成功 0 操作失败 int
data json
msg String

例子

3:修改密码

接口说明 :用户保存修改的密码时调用

请求地址 /api/v1/u/updatepwd 类型
请求参数 appUserId 当前用户Id String
orginalPwd 原始密码 String
newPwd 新密码 String
返回值 result 1 操作成功 0 操作失败
data json

例子

4:忘记密码

接口说明 :忘记密码，保存新的密码时调用

请求地址 /api/v1/u/forgetpwd 类型
请求参数 mobile 手机号 String
identifyCode 手机短信验证码 String
newPwd 新密码 String
返回值 result 1 操作成功 0 操作失败 int
data json
