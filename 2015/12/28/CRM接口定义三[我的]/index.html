发表于 2015-12-28   |  

1:圈子信息列表

接口说明 :获取圈子信息列表-带5条评论

请求地址 /api/v1/circle/list 类型
请求参数 page 当前页 int
请求参数 pagenum 一页包含的数据条数 int
返回值 result 1 操作成功 0 操作失败 int
data json

3:圈子信息发表评论

接口说明 :圈子信息发表评论

请求地址 /api/v1/circle/comment/post 类型
请求参数 postid 主贴id int
请求参数 commentid 评论id int
请求参数 cotent 评论内容 string
返回值 result 1 操作成功 0 操作失败 int
