CRM APP接口定义四[消息]

发表于 2015-12-24   |  

1:获取融云token

接口说明 :获取融云token
若当前用户启用聊天时，没有从服务器得到自己的token时，调用此接口获取token（这个*暂时不用）（用户一注册即为他生成了融云token，登陆成功后，在用户信息里）

请求地址 /api/v1/rongCloud/getToken 类型
返回值 result 1 操作成功 0 操作失败 int
data json

例子

2．添加好友

请求地址 /api/v1/rongCloud/addFriends 类型
请求参数 friendid 好友Id string
请求参数 orgid 组织id string
返回值 result 1 操作成功 0 操作失败 int
data json

3．接受或者拒绝好友请求

请求地址 /api/v1/rongCloud/reRequest 类型
请求参数 friendid 接受某id的请求 string
请求参数 orgid 组织id string
请求参数 status 操作，1接受，2拒绝 string
返回值 result 1 操作成功 0 操作失败 int
data json

4．获取好友列表

当前登陆用户获取好友列表时调用

请求地址 /api/v1/rongCloud/friendsList 类型
请求参数 orgid 组织id string
返回值 result 1 操作成功 0 操作失败 int
data json
